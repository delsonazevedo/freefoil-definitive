Freefoil

Essa é uma ferramenta online que gera credenciais Oauth Google para o tinfoil no switch.

Esse processo funciona para usuários de Windows / iOS / Android / Linux.
Você deve ter o credentials.json.
Como conseguir o arquivo credential.json ?
Primeiro, faça login em:
Google Cloud Console

Crie seu projeto, configure sua credencial e faça download do arquivo credentials.json

*Só funciona em emails @gmail.com


Projeto baseado no código fonte de Sphynxintor. Designed by Ultimatum.

Gere suas credenciais!

Siga os passos abaixo.

Passo 1

Primeiro você precisará fazer upload do arquivo credentials.json.
Clique no botão“Escolher arquivo” e procure pelo arquivo no seu dispositivo.

*Uma vez carregado o seu arquivo, os campos abaixo irão ser auto-preenchidos. Tenha certeza de fazer o upload do arquivo corretamente.

Passo 2

Tenha certeza de estar conectado na conta que você deseja autenticar.

Clique no botão "Autorizar" abaixo. Isso irá abrir um janela do Google Oauth. Siga os passos e autorize o app. No final desse passo você terá uma tela de erro de carregamento “Não é possível acessar esse site” isso é completamente normal. Siga o próximo passo

Passo 3

Copie a URL e cole no campo abaixo.

Cole sua URL aqui:

Passo 4

Clique em “Download” para fazer download dos seus arquivos.

Passo 5

Copie os arquivos credentials.json, gdrive.token e token.pickle para a pasta:
SDCARD/switch/tinfoil

Passo 6

Abra o tinfoil e adicione as informações da loja, caso ainda não tenha feito.

Caso tenha algum problema com o guia, basta nos avisar na nossa comunidade.
